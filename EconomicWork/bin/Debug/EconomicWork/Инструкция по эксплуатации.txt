В папке с программой обязательно должен лежать текстовый файл "memory.txt".
Крайне не рекомендуется изменять текстовый файл "memory.txt" - программа перестанет корректно работать.
В случае, если файл "memory.txt" каким то образом повредился и/или программа выдает ошибку, ссылаясь на 
вышеуказанный файл - произведите замену файла по следующей инструкции:
1) Удалите поврежденный файл "memory.txt"
2) Создайте копию файла "memory_back.txt"
3) Переименуйте файл "memory_back.txt" в "memory.txt"
Если программа спрашивает и дает варианты ответа "Да" или "Нет" - необходимо
ввести с клавиатуры Да/д или Нет/н и нажать клавишу "Enter"
Не волнуйтесь, если перепутали клавиши - программа предупредит Вас об этом.

Возможные ошибки программы:
Номер ошибки: 0 - отсутсвует файл "memory.txt" (или другое имя файла)
Номер ошибки: 1956 или больше - поврежден файл "memory.txt"
